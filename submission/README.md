# Envío Global South AI Safety Hackathon

Informe de investigación (máx. **8 páginas**) para la rama **AI Security → Prompt injection & jailbreaks**.

## Generar PDF

```bash
./scripts/generate_submission_pdf.sh
# → Documentation/GlobalSouth-RAGE-Submission.pdf
```

El script falla si el PDF supera 8 páginas.

## Antes de enviar

1. Editar `GlobalSouth_RAGE_Submission.md`:
   - Sustituir `[Nombre Apellido]` y afiliaciones
   - Revisar resumen (≤150 palabras)
2. Regenerar PDF
3. Copiar contenido a la [plantilla oficial Apart](https://docs.google.com/document/d/1gDyYcOhqxKVGldyxt7a7sAUwzpmLaq-7S2rK4oJZAk0/copy) si el jurado exige ese formato exacto
4. Enviar por el botón **"Enviar tu proyecto"** en la página del hackathon

## Versión larga (referencia)

`RAGE_paper_es.md` (~9 páginas en PDF) — no usar para envío; solo referencia técnica.
